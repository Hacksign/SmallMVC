[ One Minute Start ]

1.Create directories names below whith proper read-write access privileges(eg. in /srv/http/website/). 

	$ mkdir /srv/http/website
	$ cd /srv/http
	$ mkdir /srv/http/{cache,config,controller,model,plugins,public,view}
	$ ls -l website/
	drwx------ 0 http http 4096 04-29 14:00 cache
	drwx------ 0 http http 4096 04-29 14:00 config
	drwx------ 0 http http 4096 04-29 14:00 controller
	drwx------ 0 http http 4096 04-29 14:00 model
	drwx------ 0 http http 4096 04-29 14:00 plugins
	drwx------ 0 http http 4096 04-29 14:00 public
	drwx------ 0 http http 4096 04-29 14:00 view

2.Create config.php in config directory with below content: 

	$ cat config/config.php 
	<?php
		$config['debug'] = true; // set true if you want to see error information
	?>

3.Create IndexController.php in controller directory with below content: 

	$ cat controller/IndexController.php 
	<?php
		class IndexController extends SmallMVCController{
			public function index(){
				$this->assign('info', 'from IndexController/index');
				$this->display('index_page');
			}
			public function default(){
				$this->assign('info', 'from IndexController/default');
				$this->layout('content_file', 'layout_file');
			}
		}
	?>

4.Create index_page.html content_file.html layout_file.html in view directory with below content: 

	$ cat view/*.html
	index_page.html:
	<?php
		echo $info;
	?>
	layout_file.html:
	{__LAYOUT__}
	<div id="layout_div">
		this line is from layout_file.html
		<br/>
		{__CONTENT__}
	</div>
	content_file.html:
	<div id="content_div">	
		this line is from content_div
		<br/>
		<?php echo $info ?>
	</div>

5.Create entry file index.php with below content: 

	$cat /srv/http/index.php
	<?php
		define('APPDIR', 'website'); //website is created in step one above.
		require('/home/SmallMVC/SmallMVC.php');	//change this line to your SmallMVC.php location
	?>

6.All set, you can access index/default method in IndexController.php with url below: 

	http://your-server.com/index.php/Index/index
	http://your-server.com/index.php/Index/default

[ Advanced ]

SmallMVC CURD:
 SmallMVC only support mysql database right now, and you must have mysql.so installed and loaded on your php server.
If you use SmallMVC framwork's default Model class, you must config your php's PDO support correctly.
Ok, let's start, First you should add these lines to your config/config.php(you can find this file in SmallMVC's config/config_database.php.example):

	<?php
	/**
	* database.php
	* just copy the content below to your project's config/config.php
	* and config it correctly
	*/

		$config['database']['plugin'] = 'SmallMVCModel'; // plugin for db access
		$config['database']['type'] = 'mysql';      // connection type
		$config['database']['host'] = 'localhost';  // db hostname
		$config['database']['name'] = 'dbname';     // db name
		$config['database']['user'] = 'dbuser';     // db username
		$config['database']['pass'] = 'dbpass';     // db password
		$config['database']['persistent'] = false;  // db connection persistence?
		$config['database']['charset'] = $config['charset'];  // connection charset

	?>

In the example above you use the framework's default Model class SmallMVCModel.php which lies in core/SmallMVCModel.php.
You can create a model with 'M()' method in your controller like this: 

	$ cat controller/IndexController.php 
	<?php
		class IndexController extends SmallMVCController{
			public function index(){
				$model_with_table_given = M('TableNameInYourDb');
				$result1 = $model_with_table_given->where('id >', 1)->query();
				$model_without_table_given = M();
				$result2 = $model_without_table_given->from('TableNameInYourDb')->where('id', 1)->query('one');
				//another example
				$model_with_table_given->where(array('id', 'insert_time <'), array(1, '2013-01-01'));
				$result3 = $model_with_table_given->query('all');//query('all') equals query(), this return a fetched array

				$this->assign('result1', $result1['some_field']);
				$this->assign('result2', $result2);
				$this->assign('result3', $result3);
				$this->display('index_page');
			}
		}
	?>

Use Third-Part Viewer Engine:
 just add these line to your config/config.php. 

	$config['system']['viewer'] = array('Your_Third_Part_viewer',array());

The first element of array is the name of your Third-Part Viewer's FULL NAME
And the second array is the param(s) which will be passed to your viewer class's construct method.
After configured this, setup your Third-Part-Viewer properly wich your Viewer required, and invoke its method in your controller throw '$this->methods_in_your_third_part_viewer()'.

Quick Methods
 M('ModelName'):create a model which lies model/ModelNameModel.php. If there is no ModelNameModel.php in model directory use $config['database']['plugin'] (default is SmallMVCModel.php) instead.
C('ControllerName'):create a controller object which lies controller/ControllerNameController.php, this file must be exists in controller directory or your invoke will be failed.
G('field_name'): get the field_name content in $config['field_name'].
load('file_name'): import a file with file_name if the file_name is like this A.b.c then import c.php in directory A/b/
